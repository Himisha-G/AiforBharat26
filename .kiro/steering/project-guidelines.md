# Multilingual Vendor Assistant - Project Guidelines

## Project Overview
This is a voice-first web application designed for local vendors and customers to communicate across languages and access city-wide price information for everyday goods.

## Core Features
1. **Multilingual Chat + Voice Communication**
   - Support for Hindi, English, and Tamil
   - Voice input/output capabilities
   - Real-time translation
   - Minimal UI for non-technical users

2. **City-Wide Price Information System**
   - Standardized city-level prices for common items
   - Voice/text queries like "Aaj aloo ka rate kya hai?"
   - Simple, conversational responses
   - No complex analytics or charts

## Technical Constraints
- Voice-first interaction prioritized
- Lightweight and fast responses
- Minimal learning curve
- End-to-end functionality over UI polish
- Real-world usability for busy markets

## Development Guidelines
- Keep interactions simple and natural
- Prioritize speed over features
- Focus on the two core features only
- Avoid adding unnecessary complexity
- Test with voice interactions primarily

## Price Data Structure
Items should include common vegetables, fruits, and grains with city-wide average prices in INR per kg format.
